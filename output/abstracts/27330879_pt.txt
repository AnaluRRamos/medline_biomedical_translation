Projeto bem sucedido de materiais e processos de fabricação exige a disponibilidade de dados fiáveis materiais. Ligas comerciais geralmente contêm um grande número de elementos, e os dados necessários para a concepção de novos materiais e processos raramente estão disponíveis. O CALPHAD (cálculo da fase Diagramas) método permite o desenvolvimento de bases de dados termodinâmicos e de propriedade, que em conjunto com métodos de extrapolação das descrições de sistemas binários e ternários aos sistemas de ordem superior, permitir o cálculo de dados para sistemas de ordem superior. Os resultados obtidos a partir de cálculos CALPHAD foram mostrados para ser valiosa para a concepção de novos materiais. Esta revisão apresenta uma visão geral do método CALPHAD, ferramentas de software e bancos de dados e dá exemplos de sua aplicação.
